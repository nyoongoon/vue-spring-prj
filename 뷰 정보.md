## cf) @RequestAttribute
```
request.setAttribute("key", value);
==> 
@RequestAttribute("key")
```
- Request 환경에서 어떤 값을 키밸류 형태로 넣어주면, Controller @RequestAttribute를 통해서 값을 받아올 수 있음.
